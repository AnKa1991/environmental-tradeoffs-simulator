# Instrukcja dla studentów

## Symulator kompromisów środowiskowych

Aplikacja służy do przećwiczenia podejmowania decyzji środowiskowych w sytuacjach, w których trzeba pogodzić kilka kryteriów jednocześnie: technicznych, środowiskowych, społecznych i ekonomicznych.

## Jak uruchomić aplikację

Jeśli aplikacja została opublikowana jako strona internetowa, wystarczy otworzyć link przekazany przez prowadzącą.

Jeśli aplikacja została przekazana jako folder ZIP:

1. Rozpakuj folder ZIP.
2. Otwórz plik `index.html` w przeglądarce internetowej.
3. Wybierz jeden z modułów i zmieniaj parametry scenariusza.
4. Przeanalizuj wynik, ostrzeżenia i ocenę wielokryterialną.

## Moduły

Aplikacja zawiera trzy moduły:

- Hałas
- Modernizacja źródła energii
- ITPOK / spalarnia odpadów

## Raport PDF

Po zakończeniu pracy można wygenerować raport, korzystając z opcji drukowania w przeglądarce i wybierając `Zapisz jako PDF`.

## Ważna uwaga

Symulator ma charakter dydaktyczny i upraszczający. Wyniki służą do ćwiczeń i dyskusji, a nie do sporządzania rzeczywistych ekspertyz środowiskowych.
